# Flask-Movie-Database-Management

The Flask Movie Database Management is a web application that allows users to manage a collection of movies. It provides a user-friendly interface for adding, deleting, rating, and reviewing movies.

## Features

- Add movies: Users can add new movies to the database by filling out a form.
- Delete movies: Users have the option to remove movies from the collection.
- Rate and review movies: Users can rate movies on a scale of 1 to 10 and provide a review.
- User interface: The application provides a user-friendly interface for seamless interaction with the movie database.

## Role of the Code

The code sets up a Flask web application and handles the core functionalities of the movie database management system. It establishes a connection to a SQLite database using SQLAlchemy. The code defines routes, view functions, and form classes for different operations such as adding, deleting, rating, and reviewing movies. It uses HTML templates and Jinja2 templating to render dynamic web pages.

By running the code, users can access the application through a web browser and perform various actions on the movie collection. The code handles the backend operations, database management, form validation, and user interface rendering to provide a smooth and intuitive experience for managing the movie database.

## Requirements

To run the Flask Movie Database Management, ensure you have the following installed:

- Python 3.x: The programming language used to develop the application.
- Flask: The web framework for building the application.
- Flask extensions: Flask-WTF, Flask-Bootstrap, and Flask-SQLAlchemy.
- SQLite: The database engine for storing movie data.
